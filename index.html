Iris

GitHub | demo

Iris is a program written in the Unity game engine which is used for relational data visualization. Once imported into the program, this data can be navigated in 3D. It's the largest project I've ever worked on and it will probably never be completely finished. My long term goal is for it to be useful in the context of data oriented research, competing, complementing, and/or integrating with tools like Stata and MatLab.

Flood

GitHub | Android Marketplace

This is a clone of an old puzzle game. Basically the goal is to "flood" a board of multi-colored tiles by clicking colored buttons and slowly absorbing tiles, starting from the top left of the board. Tiles that are on the border of your flooded section are absorbed if they match the color you click. My version keeps track of high scores and has a couple hundred levels to choose from, as well as a Zen mode where the boards are randomly generated.

2048 Island

GitHub | Android Marketplace

Like everyone else and everyone else's grandmother and cat, I made a 2048 clone for Android back when it was cool. This one is completely over-engineered for what is supposed to be a simple game, but looks really good. Animated tiles, swooping cameras, a pointless island theme, timed mode. None of it really adds much to the game, except for a new game mode I came up with where you have 4 small boards that are played separately and instead of losing when the board has no moves left, it just freezes for a couple turns and comes back with a higher score multiplier. Eventually, the game ends when all 4 boards are frozen at once.

Social Experiment

Link | Paper

My girlfriend is a graduate student at Stanford University studying political science. For her field paper, I implemented an experiment that she designed to test James Fearon's theory that a long shadow of the future increases bargaining delay (if you're interested, read the paper). The website is hosted on an EC2 instance and written with the Django web framework with a postgres database backend. Note that you actually need credentials to enter, but look at the quality of that login page!

Bubbleshooter

Github | Demo

When I was a freshman at William & Mary, there were a couple people on my hall that played the facebook game "Bubble Shooter" which is an old arcade game where you have a cannon at the bottom of the screen and you angle it to shoot and knockdown colored balls. I was bad at the game, so I wrote a program that took a screenshot of the game, calculated the optimal shot position, and clicked the mouse in a loop until the game was complete. They said it was cheating. I say it was resourceful.
